# This CLI Python tool is a helpful assistant for everyday Linux system admin tasks. 
# It handles three main jobs:

# Managing Users: 
You can easily add, update, or remove users. You can even do this for many people at once using a spreadsheet. It also saves time by sending automatic emails to new users so they can set their passwords.

# Checking System Health: 
It keeps a close eye on your system. It tracks how much memory and processing power your computer is using. It sends an alert if anything gets too high and helps you quickly find running programs.

# Organizing Files: 
It automatically cleans up messy folders by sorting files by their type. It also acts as a watchdog because it alerts you to major errors and tracks exactly when important files are changed or deleted.
